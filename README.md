# 📚 Toko Buku Bekas Online 

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
>[Masukan badge tech stack di sini jika ada, contoh: PHP, Laravel, React, Node.js]

Aplikasi e-commerce berbasis web/mobile untuk melakukan transaksi jual-beli buku bekas. Platform ini mempertemukan pecinta buku untuk mendaur ulang bacaan mereka dengan fitur pencarian, kategori buku, kondisi fisik buku, hingga manajemen transaksi yang aman.

---

## ✨ Fitur Utama

*   **Autentikasi Pengguna:** Registrasi, login, dan manajemen profil (Pembeli & Penjual).
*   **Katalog Buku Bekas:** Filter pencarian berdasarkan judul, penulis, kategori, dan **kondisi buku** (misal: Sperti Baru, Layak Baca, Coretan Sedikit).
*   **Sistem Keranjang & Checkout:** Kemudahan dalam memilih beberapa buku sekaligus.
*   **Manajemen Stok Penjual:** Dashboard khusus penjual untuk mengunggah foto detail buku, menentukan harga, dan deskripsi minus buku.
*   **Integrasi Pengiriman & Pembayaran:** [Sebutkan jika menggunakan RajaOngkir, Midtrans, dll.]
